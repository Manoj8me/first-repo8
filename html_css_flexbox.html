responsiveness using meta tag and media queries

1.here we have used meta tag

2.meta tag is used to to automatically adjust and fit the font-size,the width,scale according to the different devices like mobile,tablet,laptop these are the things that will be done automatically by meta tag

NOTE:META TAG WILL ONLY CHANGE THE FONT-SIZE AND WIDTH ACCORDING TO DIFFERENT DEVICES AND IT WONT CHANGE THE STYLES LIKE COLOR OR BACKGROUND COLOR FOR THAT WE NEED TO USE MEDIA QUERIES

3.here we have used media queries,media queries are does not change the font-size and likewise automatically to all devices,we need to manually specify what styles need to be changed according to which devices also we need to specify the device sizes and styles accordingly

so meta tag and media queries both are necessary to make a website responsive so we need to use both according to the scenarios
